JavaScript vs Java - 變數使用差異

A. 宣告方式

JavaScript

let x = 5;
const y = 10;
var z = "hello";
      

    

    

      
Java

      
int x = 5;
final int y = 10;
String z = "hello";
      

    

  







  
B. 型別轉換與賦值

  

    

      
JavaScript

      
let value = 10;
value = "ten";
      

      
      

    

    

      
Java

      
int value = 10;
value = "ten"; // 編譯錯誤
      

    

  







  
C. 區域作用域（Scope）

  

    

      
JavaScript

      
if (true) {
  let x = 1;
}
// console.log(x); // 錯誤：x 不存在
      

      
      

    

    

      
Java

      
if (true) {
  int x = 1;
}
// System.out.println(x); // 錯誤：x 不存在
      

    

  







  
D. 未初始化變數

  

    

      
JavaScript

      
let notSet;
console.log(notSet); // undefined
      

      
      

    

    

      
Java

      
int notSet;
// System.out.println(notSet); // 編譯錯誤
