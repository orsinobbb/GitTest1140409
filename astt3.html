1. Azure Speech Translation

Phrase List 格式說明
它不是翻譯詞典，而是 ASR 的「辨識提示詞」。格式上就是字串陣列；在此工具中用「一行一詞」輸入，程式會轉成 addPhrases(["AI8D","KAM10"])。 適合公司名、人名、產品名、縮寫、同音易錯詞。不要放太長的文章。

2. Azure OpenAI 4o 後處理

Speech SDK 載入中 麥克風未測 尚未開始 LLM 未測
純前端會暴露 Speech Key 與 Azure OpenAI Key，只適合本機 PoC。正式版請改成後端簽發短效 token / proxy。 若 Azure OpenAI 被瀏覽器 CORS 擋住，也需要加一層後端 proxy。
