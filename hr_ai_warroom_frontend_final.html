安全與治理提醒

⚠️ PoC 可以純前端，正式企業版不建議。
API Key 放在瀏覽器會有外洩風險，正式版應由後端代理呼叫 Azure。
正式版建議補強
  • 後端 Proxy 保護金鑰
  • 個資遮罩與敏感欄位偵測
  • 使用者權限與角色分級
  • Blob / Table / Cosmos DB 留存分析紀錄
  • Log、稽核、人工覆核狀態
