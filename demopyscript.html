📊 能力矩陣速查表

功能 可行性 效能 複雜度 建議場景
基礎 DOM 操作 ✅ 完全支援 ⭐⭐⭐⭐ 低 簡單互動、表單處理
Numpy 計算 ✅ 支援 ⭐⭐⭐ 中 資料科學 Demo、演算法驗證
背景計算 ⚠️ 需要 Worker ⭐⭐⭐⭐ 高 重計算且需保持 UI 流暢
大數據處理 ⚠️ 有記憶體限制 ⭐⭐ 高 中小型資料集視覺化
複雜 UI 框架 ❌ 不建議 ⭐ 極高 用 React/Vue 替代
即時通訊 ⚠️ 透過 JS ⭐⭐⭐ 高 簡單 WebSocket，複雜邏輯用 JS
機器學習推論 ✅ 支援 ⭐⭐ 中 小模型推論、教學展示
⚠️ 測試前注意事項
  • 首次載入慢：PyScript + 套件可能需要 5-10 秒載入
  • 記憶體消耗：部分測試會佔用大量記憶體（建議關閉其他分頁）
  • 瀏覽器差異：Chrome/Edge 效能最佳，Safari 可能有相容性問題
  • 行動裝置：不建議在手機上測試高負載項目（02、03、04）

💡 測試目標

這個套件的目的不是證明「PyScript 很厲害」或「PyScript 不夠好」， 而是讓你清楚知道在什麼情況下該用 PyScript，什麼情況下該用 JavaScript。
